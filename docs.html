📘 NanoJS - Documentação Básica

Este guia explica como usar o framework NanoJS com base no projeto fornecido.

🚀 Inicialização da Aplicação

O arquivo principal main.js configura as rotas e inicia o router:

import { Router } from 'nanojs';
import { Home } from './pages/Home.js';
import { UserDetail } from './pages/UserDetail.js';

const routes = [
  { path: '/', component: Home },
  { path: '/user/:id', component: UserDetail }
];

const router = new Router(routes, '#app');
router.init();

Explicação:

  • Router: Gerencia navegação SPA
  • routes: Define caminhos e componentes
  • #app: Elemento onde tudo será renderizado

🧩 Criando Componentes

Componentes herdam da classe Component:

import { Component } from 'nanojs';

export class Home extends Component {
  constructor(props) {
    super(props);

    this.state.name = 'Mestre do JS';
    this.state.count = 0;
  }
}

State

O state é reativo — ao alterar valores, a UI atualiza automaticamente.

🎨 Renderização Declarativa

O método render() retorna objetos JS que representam HTML:

render() {
  return {
    div: {
      content: [
        { h1: { content: this.state.title } },
        { p: { content: 'Texto exemplo' } }
      ]
    }
  };
}

Regras:

  • Tags HTML são objetos (div, p, etc)
  • content define conteúdo interno
  • Pode ser string, array ou outro objeto

⚡ Eventos

Eventos são definidos diretamente como funções:

{
  button: {
    content: 'Clique',
    onclick: () => alert('Clicou!')
  }
}

Exemplo com input:

{
  input: {
    type: 'text',
    value: this.state.newTask,
    oninput: e => this.state.newTask = e.target.value
  }
}

📌 Manipulação de Lista

Use map() para renderizar listas:

{
  ul: {
    content: this.state.tasks.map(task => ({
      li: { content: task.title }
    }))
  }
}

➕ Exemplo: Adicionar Task

addNewTask() {
  if (this.state.newTask.trim() === '') return;

  const newTask = {
    id: Date.now(),
    title: this.state.newTask
  };

  this.state.tasks = [...this.state.tasks, newTask];
  this.state.newTask = '';
}

Isso atualiza automaticamente a interface.

🔗 Navegação SPA

Use data-link para navegação sem reload:

{
  a: {
    href: '/user/37',
    content: 'User page',
    'data-link': true
  }
}

👤 Parâmetros de Rota

Rotas com parâmetros:

{ path: '/user/:id', component: UserDetail }

Capturando o parâmetro:

constructor(props) {
  super(props);
  this.state.userId = props.params[0];
}

📄 Página de Detalhes

render() {
  return {
    div: {
      content: [
        { h1: { content: 'Detalhes do Usuário' } },
        {
          p: {
            content: [
              'ID: ',
              { b: { content: this.state.userId } }
            ]
          }
        }
      ]
    }
  };
}

✅ Conclusão

  • Framework leve e declarativo
  • Sem HTML direto — tudo via JS
  • State reativo automático
  • SPA com roteamento integrado
